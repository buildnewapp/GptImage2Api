# 关于 SdanceAI

SdanceAI 是 Seedance 2.0 的在线体验平台与 API 服务提供商。我们致力于让用户体验高质量的 AI 视频生成，并在视频中实现原生音频同步。同时，我们也为开发者提供稳定的 API 接入服务，助力将 AI 视频能力集成到更多应用中。

## 我们的故事

这是一个关于激情与创造的故事。

SdanceAI 的创始人 Paidax 是一位致力于技术的开发者，同时也是一位充满爱心的父亲（奶爸）。在一段难得的假期时光里，他没有选择单纯的休息，而是决定利用这段时间，将自己对 AI 视频生成技术的构想变为现实。

作为一个奶爸，Paidax 深知时间的宝贵和创造力的重要性。他希望通过 SdanceAI，不仅仅是展示 Seedance 2.0 的强大功能，更是为了证明：即使是在忙碌的生活节奏中，只要有梦想和行动，利用碎片化的时间也能创造出令人惊叹的产品。

这个项目的诞生，源于对技术的热爱，也源于对未来无限可能的期许。我们希望 SdanceAI 能成为连接您与 AI 视频创作的桥梁。

## 我们的使命

我们的使命很简单：**让 AI 视频创作变得触手可及**。

无论是普通用户想要体验 Seedance 2.0 带来的震撼效果，还是开发者希望通过 API 构建自己的 AI 应用，SdanceAI 都将为您提供最坚实的支持。
